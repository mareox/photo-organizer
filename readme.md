# Photo Organizer

Python tool for organizing and managing photo collections.

## Features

- Organize photos by date
- Duplicate detection
- Metadata extraction
- Batch renaming

## Requirements

- Python 3.x
- See requirements.txt for dependencies

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python organize_photos.py [source_dir] [destination_dir]
```
